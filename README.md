# Electro-Fih
Egy elektromosságot megfigyelő program ami a felhasználó meglévő eszközeinek a Watt-ban mért teljesitményét méri egy SQL adatbázist felhasználva.
<img width="778" height="778" alt="Electro-Fih" title="Electro-Fih" src="https://github.com/user-attachments/assets/c7eaad46-ae2a-4a2d-8cc4-b5be2472fbbf"/>

Használati Útmutató:

1. Töltsd le a legújabb verziót a Releases-nél, a Zip-et és csomagold ki egy tetszőleges mappában.

2. A(z) .exe fájl-t kell megnyitni.

3. A programban kell hogy csinálj egy fiókot hogy tudd használni a funkcióit.
Itt az Email és a Jelszó rublikát kell elsőnek kitölteni, és utána rá kattintani a Regisztráció gombra. 

4. A háztartásban lévő eszközöket a ház hozzáadása után tudod megtenni.

5. A bal olali sávban meg fog jelenni a ház és ott lehet a használatban lévő eszközöket hozzá rendelni.
